# FacePass

## Quick Start

1. Install dependencies:

```bash
pip install -r requirements.txt
```

1. Run the application:

```bash
python main.py
```

1. Open <http://localhost:8000> in your browser

## Alternative: Run Demo (OpenCV window)

```bash
python demo.py
```
